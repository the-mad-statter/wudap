## Overview

The R package `wudap` is a simplistic LDAP client for Washington
University.

<br />

## Installation

You can install `wudap` from
[GitHub](https://github.com/the-mad-statter/wudap) with:

``` r
pak::pkg_install("the-mad-statter/wudap")
```
